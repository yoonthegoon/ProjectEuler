# Project Euler

A [website](https://projecteuler.net/) dedicated to a series of computational
problems intended to be solved with computer programs.

## About

This repo hosts my solutions to these problems in [Go](https://go.dev/).  
If you wish to verify any of my solutions, simple run

```sh
go run problems/problem0001/main.go
```

replacing 0001 with the number of the problem you'd like to run padding with 0s
to width 4.
